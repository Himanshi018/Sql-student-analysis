# SQL Mini Project — Student Course Analysis

This is a simple SQL project demonstrating:
- Database schema design
- Table creation
- Data insertion
- Analytical SQL queries (JOIN, GROUP BY, Aggregates)

## Files
- `schema.sql` — Creates all tables
- `insert_data.sql` — Inserts sample data
- `queries.sql` — Demonstrates analysis queries

## Concepts Used
- Primary & Foreign Keys
- JOINs (INNER JOIN)
- GROUP BY
- ORDER BY
- Aggregate Functions (COUNT, AVG)
- CASE WHEN (for grade conversion)

## How to run
1. Create database:  
   `CREATE DATABASE studentdb;`

2. Run schema:  
   `\i schema.sql`

3. Insert data:  
   `\i insert_data.sql`

4. Run queries:  
   `\i queries.sql`

This is a learning project for demonstrating SQL fundamentals in interviews.
